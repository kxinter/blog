# 1 dockers开启privileged 

解决centos7容器无法使用systemctl命令的问题
```bash
    docker run -d -e "container=docker" --privileged=true -v /sys/fs/cgroup:/sys/fs/cgroup --name centos7 centos:7.5 /usr/sbin/init
```
