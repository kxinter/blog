# 安装expect命令
<!--more-->
``` bash
yum install expect -y
```
**示例脚本**

``` vim docker自动登陆脚本
#！/usr/bin/expect
set timeout 2                   # 超时时间2s
spawn docker login 192.168.0.200:5000   # 要执行的命令
expect "Username"                # 匹配字符Username
send "user\r"                  # 键入用户user \r 回车
expect "password"               # 匹配password
send "pass\r"                 # 键入密码pass \r 回车
interact
```
